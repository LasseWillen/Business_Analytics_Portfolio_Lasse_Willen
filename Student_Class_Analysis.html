Exploring University Class Dataset

As a student, I find it interesting to explore the correlation between different factors in university. For instance, have you ever wondered if your grades are related to any other factors? The following analysis presents insights from a university class dataset.

Data Cleaning in Python

To analyze the dataset, it first needed to be cleaned, as there were duplicate data rows and outliers. Also, a part of the task in analyzing the data was to find values for five missing values marked as xxxxx in the Excel sheet.

For an in-depth analysis, view the Python notebook here: Python Analysis Notebook.

Correlation Matrix Visualization Code and Visualization

The following code is a snippet from the analysis and visualizes the correlation between the numeric categories of the dataset. Please find the entire Python analysis above.

# show correlation matrix
import seaborn as sns
import matplotlib.pyplot as plt

# Select columns with numeric datatypes only
df_numeric = data_cleaned_filtered.select_dtypes(include=[np.number]) # do this so that only numeric data types are used

correlation_matrix = df_numeric.corr()
# Design the correlation matrix
plt.figure(figsize=(10, 8))
sns.heatmap(correlation_matrix, annot=True, fmt=".2f", cmap='coolwarm') # coolwarm for coloring
plt.show()

Further Analysis in Excel

After the analysis in Python, further analysis in Excel was made. Using Excel, I started to calculate the five missing values (xxxxx) in the spreadsheet. After that, the dataset demographics tab of the dataset was merged with the second tab “personalities” and cleaned again in Excel.

In the dataset, the first tab shows the first tab after the Python data cleaning.
